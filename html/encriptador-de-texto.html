Encriptador de texto

Durante mi formación como desarrollador frontend en Alura, creé este encriptador de texto como parte de un proyecto práctico. La aplicación permite a los usuarios encriptar y desencriptar texto utilizando diferentes algoritmos. Utilicé HTML, CSS y JavaScript para desarrollar la interfaz y la lógica de encriptación. Este proyecto me permitió aprender sobre seguridad informática y algoritmos de encriptación, además de mejorar mis habilidades en el desarrollo web frontend. Fue una experiencia educativa y práctica que contribuyó significativamente a mi crecimiento como desarrollador.
