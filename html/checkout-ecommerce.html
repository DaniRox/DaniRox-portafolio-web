Checkout E-commerce

Este proyecto fue creado como parte de un desafío en DevChallenge para diseñar una página de checkout de e-commerce. La página está diseñada para proporcionar una experiencia de compra fluida y eficiente para los usuarios. Incluí características como un resumen detallado de la compra, opciones de envío personalizables y un formulario de pago seguro. Utilicé HTML, CSS y JavaScript para desarrollar la página, asegurándome de que sea responsive y funcione correctamente en diferentes dispositivos. Me enfoqué en crear un diseño limpio y fácil de entender, centrándome en la usabilidad y la accesibilidad. ¡Fue un proyecto emocionante que me permitió mejorar mis habilidades en el diseño y desarrollo web!
